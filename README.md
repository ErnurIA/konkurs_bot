# konkurs_bot

Telegram-бот для проведения онлайн-олимпиад и тестов для школьников (1–11 классы).

## Возможности
- Выбор класса участника
- Ввод ФИО (для диплома)
- Тест в формате Telegram Quiz
- Таймер на каждый вопрос (90 секунд)
- Автоматический переход к следующему вопросу
- Кнопка пропуска вопроса
- Подсчёт правильных ответов
- Автоматическое определение результата (диплом / сертификат)

## Стек технологий
- Python 3.13
- aiogram 3.x
- Telegram Bot API

## Структура проекта
- `bot.py` — точка входа, запуск бота
- `flow.py` — логика диалога (класс, ФИО, старт теста)
- `quiz.py` — логика теста, таймер, автопереход
- `questions.py` — база вопросов
- `.env` — токен Telegram-бота (не хранится в репозитории)
